Restaurant Service API

About

This service provides an API for restaurant sellers to manage their dishes and view order history in the online homemade food ordering system.

Available Endpoints

  • POST /api/auth/login - Authenticate restaurant seller
  • GET /api/restaurants/{restaurantId}/dishes - Get all dishes for a restaurant
  • POST /api/restaurants/{restaurantId}/dishes - Add a new dish
  • PUT /api/restaurants/{restaurantId}/dishes/{dishId} - Update a dish
  • DELETE /api/restaurants/{restaurantId}/dishes/{dishId} - Delete a dish
  • GET /api/restaurants/{restaurantId}/orders - Get order history

Testing

Access /api/bootstrap/init to initialize test data for development purposes.
